Konfiguracja Firebase

Otwórz plik w Notatniku i zastąp blok firebaseConfig swoimi danymi.

  1. Wejdź na console.firebase.google.com
  2. Projekt → Dodaj aplikację webową (</>) → skopiuj firebaseConfig
  3. Firestore Database → Create database → Test mode → europe-west1
  4. Wklej config do pliku, wrzuć na netlify.com/drop
